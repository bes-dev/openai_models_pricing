API Usage

Access pricing data via JSON API:

Simple API (recommended)

GET https://<your-github-pages-url>/api.json 
{
  "models": { ... },
  "timestamp": "2025-01-27T12:00:00Z",
  "models_count": 20,
  "source": "openai_official_pricing_page"
}

Full Data

GET https://<your-github-pages-url>/pricing.json

Historical Data (90 days)

GET https://<your-github-pages-url>/history.json

JavaScript Example

fetch('./api.json')
  .then(res => res.json())
  .then(data => {
    console.log('Models:', data.models);
    console.log('Updated:', data.timestamp);
  });

Python Example

import requests

url = 'https://<your-github-pages-url>/api.json'
data = requests.get(url).json()

for model_name, model_data in data['models'].items():
    print(f"{model_name}: {model_data}")
